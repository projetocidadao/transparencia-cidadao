# Módulo Fiscalização Cidadã

## Área de Mídia

### O que o cidadão pode enviar

| Tipo | Formato | Tamanho máximo | Para que serve |
|------|---------|-----------------|----------------|
| 📷 Foto | JPG, PNG | 10MB | Comprovar problema, situação atual |
| 🎥 Vídeo | MP4, MOV | 100MB | Mostrar o problema em movimento |
| 📄 Documento | PDF, JPG | 10MB | Comprovantes, relatórios, notas |

### Como funciona o upload

```
1. Clica em "Adicionar prova"
2. Escolhe tipo: foto, vídeo ou documento
3. Faz upload ou tira foto na hora
4. Sistema salva com data/hora automática
5. Prova fica vinculada à demanda
```

### Validações automáticas

- Data/hora da mídia é registrada automaticamente
- Localização da mídia (GPS) quando possível
- Formato verificado antes do upload
- Compressão automática de fotos/vídeos para economizar espaço

## Guia do Cidadão Fiscal

### O que é ser um fiscal cidadão?

É ser os "olhos do governo" — acompanhar de perto se o dinheiro público está sendo bem usado, se as obras estão sendo feitas direito, se os serviços estão funcionando.

### Por que é importante?

- Mais olhos vendo = menos desperdício
- O cidadão conhece a realidade local
- Ajuda o governo a resolver problemas mais rápido

---

## Passo a Passo para Fiscalizar

### Passo 1: Escolha o que fiscalizar

- Uma obra pública (escola, hospital, estrada)
- Um serviço (coleta de lixo, transporte)
- Uma entrega (material, equipamento)

### Passo 2: Observe com atenção

- Como está a qualidade do serviço?
- Está no prazo?
- O dinheiro está sendo bem usado?
- Tem algo errado?

### Passo 3: Documente

- Tire fotos do antes, durante e depois
- Grave vídeos se possível
- Anote datas, nomes, valores
- Guarde comprovantes

### Passo 4: Envie pelo sistema

- Acesse "Nova Fiscalização"
- Escolha o tipo
- Preencha os dados
- Anexe suas provas (fotos, vídeos, documentos)
- Envie

### Passo 5: Acompanhe

- Você receberá um código
- Use para acompanhar o andamento
- Receba notificações sobre respostas

## Dicas de Como Fiscalizar Bem

### 📸 Fotos boas

- Tire várias fotos
- Mostre o contexto (rua, prédio)
- Tire foto de perto dos detalhes
- Fotografe data/hora se possível

### 🎥 Vídeos úteis

- Grave em movimento mostrando o todo
- Pause em detalhes importantes
- Fale o que está vendo (narração)
- Duração: 30 segundos a 2 minutos é ideal

### 📄 Documentos importantes

- Notas fiscais
- Contratos
- Relatórios
- Qualquer documento oficial

## Acessibilidade

### Para todos os perfis

| Perfil | Como atendemos |
|--------|----------------|
| 👴 Idoso | Texto grande, botões grandes, voz guiding |
| 👁️ Cego | Leitor de tela, comandos de voz |
| 🦽 Cadeirante | Interface adaptável, acessibilidade física |
| 📱 Básico | Poucos cliques, ícones claros |
| 💻 Avançado | Mais opções, filtros, exportação de dados |

### Recursos de acessibilidade

- **Tamanho de texto ajustável**
- **Alto contraste** (tela escura/clara)
- **Comandos de voz**
- **Leitor de tela compatível**
- **Navegação por teclado**
- **Tradução em Libras** (futuro)

## Níveis de Participação

### Nível 1: Observador

- Apenas acompanha demandas
- Vê o dashboard público
- Não precisa se identificar

### Nível 2: Contribuidor

- Cria demandas simples
- Envia fotos
- Não precisa se identificar

### Nível 3: Fiscal

- Documenta obras/serviços
- Envia relatórios
- Pode se identificar ou não

### Nível 4: Auditor

- Tem acesso a mais dados
- Pode fazer perguntas formais
- Recebe notificações prioritárias

---

## Resumo

O sistema permite que qualquer cidadão, sem conhecimento técnico, possa:

1. **Observar** — Ver o que está acontecendo
2. **Documentar** — Provar com fotos, vídeos, documentos
3. **Reportar** — Enviar de forma simples
4. **Acompanhar** — Ver o andamento
5. **Fiscalizar** — Garantir que o dinheiro público está sendo bem usado

O importante é a participação. O sistema se adapta ao nível de cada cidadão.
