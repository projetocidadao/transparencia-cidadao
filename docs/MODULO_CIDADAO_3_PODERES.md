# Módulo Cidadão e os 3 Poderes

## Visão Geral

O Módulo Cidadão é a porta de entrada do sistema. É por onde o cidadão interage diretamente, criando demandas, acompanhando status e recebendo notificações.

## Como funciona a conexão

```
Cidadão → Módulo Cidadão → Sistema → 3 Poderes
```

O cidadão não precisa saber qual poder está envolvido — ele faz uma demanda e o sistema direciona automaticamente.

---

## Executivo (Prefeituras, Governos Estaduais)

### O que conecta
- Demandas de serviços (ruas, iluminação, saneamento)
- Obras públicas (escolas, hospitais, estradas)
- Licitações e contratos
- Execução orçamentária

### Fluxo
1. Cidadão cria demanda: "Rua X com buraco"
2. Sistema identifica como competência do Executivo (Prefeitura)
3. Demanda é direcionada ao órgão responsável
4. Cidadão acompanha o status
5. Notificação quando houver resposta

---

## Legislativo (Câmaras de Vereadores, Assembleias, Congresso)

### O que conecta
- Projetos de lei
- Votações
- Presenças dos deputados/vereadores
- Gastos da câmara
- Diárias e viagens

### Fluxo
1. Cidadão pode acompanhar projetos de lei
2. Verificar como cada representante votou
3. Acompanhar gastos públicos da câmara
4. Receber alertas sobre votações importantes

---

## Judiciário (TJ, STF, Tribunais)

### O que conecta
- Processos públicos (transparência judicial)
- Sentenças de interesse público
- Estatísticas de julgamento
- Prazos processuais

### Fluxo
1. Sistema busca processos públicos
2. Cidadão pode acompanhar casos de interesse público
3. Ver estatísticas de julgamento
4. Receber alertas sobre decisões importantes

---

## Resumo

| Poder | Dados disponíveis | Ação do cidadão |
|-------|-------------------|-----------------|
| Executivo | Obras, serviços, licitações, gastos | Criar demandas, acompanhar |
| Legislativo | Leis, votações, gastos de deputados | Acompanhar, verificar |
| Judiciary | Processos, sentenças, estatísticas | Acompanhar casos |

---

## Objetivo

Permitir que o cidadão comum tenha acesso simples e direto às informações dos 3 poderes, sem precisar entender a estrutura do governo. Ele faz uma pergunta/demanda e o sistema mostra a resposta.
